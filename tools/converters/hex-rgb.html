فرمت‌های دیگر رنگ

راهنمای استفاده

کد رنگ HEX چیست؟

کدهای HEX روشی برای نمایش رنگ‌ها با استفاده از سیستم اعداد پایه 16 هستند. هر رنگ با ترکیبی از 6 کاراکتر (0-9 و A-F) که به صورت #RRGGBB نمایش داده می‌شوند، مشخص می‌شود.

مقادیر RGB چیست؟

RGB مخفف Red, Green, Blue است و هر رنگ با سه مقدار بین 0 تا 255 مشخص می‌شود که شدت هر یک از رنگ‌های اصلی را نشان می‌دهد.

نحوه استفاده از ابزار:

  1. برای تبدیل HEX به RGB، کد HEX را وارد کرده و دکمه "HEX به RGB" را بزنید
  2. برای تبدیل RGB به HEX، اسلایدرها را تنظیم کرده و دکمه "RGB به HEX" را بزنید
  3. می‌توانید با دکمه "رنگ تصادفی" یک رنگ تصادفی ایجاد کنید
  4. برای کپی کردن هر فرمت رنگی از دکمه‌های مربوطه استفاده کنید

نمونه کدهای رنگ:
