HTML Entities چیست؟

HTML Entities روشی برای نمایش کاراکترهای خاص در HTML است که ممکن است با تگ‌های HTML اشتباه گرفته شوند یا در مرورگرها به درستی نمایش داده نشوند. هر Entity با علامت آمپرسند (&) شروع و با سمیکالن (;) پایان می‌یابد.

نحوه استفاده از ابزار:

  1. متن یا کد HTML خود را در کادر "متن اصلی" وارد کنید
  2. برای کدگذاری از دکمه "کدگذاری" یا "کدگذاری همه" استفاده کنید
  3. برای رمزگشایی، متن کدگذاری شده را در کادر ورودی قرار داده و دکمه "رمزگشایی" را بزنید
  4. می‌توانید با تنظیمات موجود، نحوه کدگذاری را شخصی‌سازی کنید
  5. برای استفاده از Entities پرکاربرد می‌توانید از جدول پایین صفحه استفاده کنید

تفاوت انواع کدگذاری:

  • Entity Name: &name; (مثال: &copy; برای ©)
  • Entity Number: &#number; (مثال: &#169; برای ©)
  • Hex Code: &#xhex; (مثال: &#xA9; برای ©)

کاربردهای HTML Entities:

  • جلوگیری از تفسیر نادرست کاراکترهای خاص به عنوان تگ HTML
  • نمایش کاراکترهایی که در صفحه‌کلید وجود ندارند
  • پیشگیری از حملات XSS (Cross-Site Scripting)
  • نمایش صحیح کاراکترهای خاص در تمام مرورگرها
