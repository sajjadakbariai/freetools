راهنمای استفاده

1. فرمت کردن JSON: کد JSON خود را در کادر ورودی وارد کرده و روی دکمه "فرمت کن" کلیک کنید.

2. فشرده کردن JSON: برای حذف فاصله‌ها و خطوط جدید از دکمه "فشرده کن" استفاده کنید.

3. اعتبارسنجی: دکمه "اعتبارسنجی" ساختار JSON را بررسی می‌کند.

4. نمونه JSON: برای تست ابزار می‌توانید از دکمه "نمونه JSON" استفاده کنید.

نمونه JSON معتبر:

{
  "name": "ToolsFree",
  "url": "https://toolsfree.ir",
  "tools": ["JSON Formatter", "Base64 Encoder", "JWT Decoder"],
  "stats": {
    "visitors": 10000,
    "rating": 4.8
  }
}
